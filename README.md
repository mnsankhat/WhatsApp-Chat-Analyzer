# 💬 WhatsApp Chat Analyzer

A Data Analysis + NLP based web application that provides insights from WhatsApp chat data.

---

## 🚀 Features

- Analyze total messages, words, media, and links
- Monthly and daily chat timeline
- Most active users in group
- WordCloud visualization
- Most common words analysis
- Emoji usage analysis
- Activity heatmaps (daily & monthly)

---

## 🧠 Concepts Used

- Data Cleaning & Preprocessing
- Natural Language Processing (NLP)
- Data Visualization
- Exploratory Data Analysis (EDA)

---

## 🛠️ Tech Stack

- **Language:** Python
- **Libraries:** Pandas, Matplotlib, Seaborn, WordCloud, Emoji
- **Frontend:** Streamlit
- **Tools:** Jupyter Notebook, Git, GitHub

## ⚙️ How It Works

1. Upload exported WhatsApp chat file (.txt)
2. Data preprocessing:
   - Extract user, message, date
   - Clean text data
3. Perform analysis:
   - Message statistics
   - Timeline trends
   - User activity
4. Generate visual insights:
   - Graphs, charts, wordcloud, emoji analysis
